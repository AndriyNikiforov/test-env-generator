# Test env generator

TEG is a tool for generate skeleton for fast start write tests.

## Installation

```npm install test-env-generator -D ```

## Usage

* Run generator with path where must be unzip ``` teg <option> <path> ```
* Run help comand ``` teg -h ```

Options
* selenium
* appium
* webdriver.io
